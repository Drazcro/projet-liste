Fonctionnement de l'API

Envoyer une requête HTTP à l'adresse http://api-liste.alwaysdata.net/api/v1. L'API est de type REST c'est à dire que vous devrez envoyer la la requête avec la bonne méthode :
  • GET Récupérer des information, l'id dans l'url
  • POST Ajouter de l'information, informations en paramètre
  • PUT Modifier de l'information, l'id de l'entités concernées dans l'url, les informations en paramètre
  • DELETE Supprimer de l'information, l'id dans l'url

Liste des utilisations possible de l'API

  • Gestion des utilisateurs

    • Interactions possibles

    • GET /utilisateurs
      Selectionner tout les utilisateurs
    • GET /utilisateurs/{id}
      Selectionner un utilisateur par id
    • GET /utilisateurs/{pseudo}/{password}
      Selectionner un utilisateur par pseudo et mot de passe (utile pour la connexion)
      • POST/utilisateurs
        Créer un utilisateur
        PARAMS
        Nom Description Valeur
        pseudo Le pseudo de l'utilisateur VARCHAR
        password Le mot de passe NON CRYPTE de l'utilisateur VARCHAR
        permission La permission de l'utilisateur INT
        role Le rôle de l'utilisateur INT
      • PUT/utilisateurs/{id}
        Modifier un utilisateur par id
        PARAMS
        Nom Description Valeur
        pseudo Le pseudo de l'utilisateur VARCHAR
        password Le mot de passe NON CRYPTE de l'utilisateur VARCHAR
        permission La permission de l'utilisateur INT
        role Le rôle de l'utilisateur INT
    • DELETE/utilisateurs/{id}
      Supprimer un utilisateur par id
  • Gestion des listes

    • Interactions possibles

    • GET /listes
      Selectionner toutes les listes
    • GET /listes/{id}
      Selectionner une liste par id
    • GET /listes/all/{idUser}
      Selectionner tout les utilisateurs
    • GET /listes/{idListe}/elements
      Selectionner tout les élements d'une liste par id
      • POST/listes
        Créer une liste
        PARAMS
        title Description Valeur
        title Le nom de la liste VARCHAR
        description La description de la liste TEXT
        visibility La visibilité de la liste INT
        idUser L'id de l'utilisateur créateur de la liste INT
      • PUT/listes/{id}
        Modifier une liste
        PARAMS
        title Description Valeur
        title Le nom de la liste VARCHAR
        description La description de la liste TEXT
        visibility La visibilité de la liste INT
        idUser L'id de l'utilisateur créateur de la liste INT
    • DELETE/listes/{id}
      Supprimer une liste
  • Gestion des etiquettes

      • POST/etiquettes
