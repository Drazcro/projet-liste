Depôt l'API

https://github.com/Drazcro/projet-liste

Fonctionnement de l'API

Envoyer une requête HTTP à l'adresse https://projet-liste2018.000webhostapp.com/api/v1. L'API est de type REST c'est à dire que vous devrez envoyer la la requête avec la bonne méthode :
  • GET Récupérer des information, l'id dans l'url
  • POST Ajouter de l'information, informations en paramètre
  • PUT Modifier de l'information, l'id de l'entités concernées dans l'url, les informations en paramètre
  • DELETE Supprimer de l'information, l'id dans l'url

Liste des utilisations possible de l'API

  • Gestion des utilisateurs

    • Interactions possibles

    • GET /utilisateurs
    • GET /utilisateurs/{id}
    • GET /utilisateurs/{pseudo}/{password}
      • POST/utilisateurs
        PARAMS
        Nom Description Valeur
        pseudo Le pseudo de l'utilisateur VARCHAR
        password Le mot de passe NON CRYPTE de l'utilisateur VARCHAR
        permission La permission de l'utilisateur INT
        role Le rôle de l'utilisateur INT
      • PUT/utilisateurs/{id}
        PARAMS
        Nom Description Valeur
